Virtual Event Styling Examples

This topic provides some sample code that can be used to customize the Virtual Event template.

Social sharing icons

By default, if the video sharing settings are enabled for the experience, they will not appear next to videos on the video detail pages. The following CSS will display the social icons on the video detail pages.


.share-list {
  display: block !important;
}
.share-list a {
  display: inline-block !important;
}
.share-list li {
  display: block !important;
}

Site background image

The following CSS can be used to add a background image to the site.


.main-content {
   background: #152538 url(https://solutions.brightcove.com/vet/excursion-con/bg.png) no-repeat center;
}
Note: For information on the properties that can be used with the background property, see this link.

Hero background video

The following static content object can be added to the configuration file to add a background video to the site. This should replace the existing hero image in the configuration file.


"static_content": [{
      "insert_target": ".bcg-fixed-width-container",
      "insert_mode": "prepend",
      "content": "<div class='video-bg'><video loop muted autoplay class='video-bg__video'><source src='https://solutions.brightcove.com/bcls/gallery/loop.mp4' type='video/mp4'></video></div> <div class='bc-hero-section' style='padding-bottom:65px;position: relative;'><div id='hero-logo'></div></div>",
      "require_roles": [],
      "omit_roles": []
    }
