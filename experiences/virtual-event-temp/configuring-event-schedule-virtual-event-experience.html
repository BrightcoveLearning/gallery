Configuring the Event Schedule

In this topic you will learn how to customize the event schedule of a Virtual Event Experience.

The event schedule is typically used to display the scheduled live events that appear in the featured collection.

Based upon the number of days in the schedule, a Show dropdown will appear allowing the viewer to select the day's schedule to view.

The schedule information is customized by editing the schedule object in the configuration file.

  • name - Session name
  • description - Session description
  • starts_at - Session start date and time
  • ends_at - Session end date and time
  • live_id - Not used
  • vod_id - Video ID
  • speakers - Session speaker information
  • image_url - Session image. Default image size is 320px x 180px. Images will be cropped to this size.

The schedule object in the configuration file also has parameters for the title that appears above the schedule and to set whether the schedule displays in an expanded mode.
