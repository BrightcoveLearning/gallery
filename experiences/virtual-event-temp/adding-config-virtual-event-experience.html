Adding the Configuration File to the Experience

In this topic you will learn how to add the configuration file to a Virtual Event Experience.

The Virtual Event Experience requires a configuration file be included with the site. The configuration file includes site settings, layout options, event schedule, speakers and other content that will be displayed. After creating a new Virtual Event Experience, adding the configuration file is the first thing that should be done

To add the configuration file to a Virtual Event Experience, follow these steps.

  1. Go to the following link:
    https://virtual-events-experience.s3.amazonaws.com/v1/config.txt
  2. Copy the content to the clipboard.
  3. Open the Gallery module.
  4. Edit the experience.
  5. Click APPEARANCE AND BEHAVIOR > Header and Footer in the left navigation.
  6. Paste the configuration file into the Custom Header HTML field.
  7. Click Save.

Make sure you republish the experience so the changes appear.
