Customizing the Header of a Virtual Event Experience

In this topic you will learn how to customize the header of Virtual Event Experience pages.

The header will appear at the top of all the pages. The header is constructed from values in the configuration file. The header is made up of the following components:

  • Header logo
  • Links
  • Social icons
  • Search

Header logo

The header image is specified by the images object in the configuration file.

  • header_logo - URL to image to appear in the header
.Png and .jpg files are supported.

Links

By default, the HOME, LIVE STREAM and ON-DEMAND SESSIONS links are hardcoded in the template:

  • HOME - Goes to home, top of page
  • LIVE STREAM - Goes to the featured collection (top video player)
  • ON-DEMAND SESSIONS - Goes to the tracks section

Additional links can be added by editing the header_links object in the configuration file.

  • link_url - Destination URL
  • label - Label to appear in the header

Social icons

By default, the header will display social icons:

  • Facebook
  • Instagram
  • Twitter
  • LinkedIn
  • YouTube
The social icons that appear in the header can be edited/removed by editing the social object in the configuration file. For information, see Configuring the Social Icons for a Virtual Event Experience.

Search

To enable search and display the search field, enable search for the experience. By default, the search field in the header will display a single line for text entry.

The search box in the example has been styled to display a box around the search field.
