Overview of the Virtual Event Template

This topic provides a brief overview of the sections provided with the Virtual Event template. Links to other detailed topics are also provided for each section.

Header

The header will appear at the top of all the pages. The header is constructed from values in the config file. The header is made up of the following components:

  • Header logo
  • Links
  • Social icons
  • Search

For information on customizing the header, see Customizing the Header of a Virtual Event Experience.
