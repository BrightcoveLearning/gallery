Customizing the Footer of a Virtual Event Experience

In this topic you will learn how to customize the footer of Virtual Event Experience pages.

The footer will appear at the bottom of all the pages. The footer is constructed from values in the configuration file. Each footer item in the configuration file will have an associated:

  • Link URL
  • Label
  • Group

Groups are used to control where the label/link appears in the footer. There are four available groups:

  • Group 0 - Used for HTML callouts
  • Group 1 - Used for legal text and copyright information
  • Groups 2 & 3 - Used for lists of links (recommended to use a max of 5 links)

The social icons that appear in the footer can be edited/removed by editing the social object in the configuration file. For information, see Configuring the Social Icons for a Virtual Event Experience.

Adding links to the footer

Footer links are configured by editing the footer_links object in the configuration file.

  • link_url - Destination URL
  • label - Label to appear in the footer
  • group - Group the link appears in
