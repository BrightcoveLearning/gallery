Creating a Virtual Event Experience

In this topic you will learn about creating a Virtual Event Experience.

Gallery provides a Virtual Event template which can be used to create Virtual Event Experiences for hosting and delivering virtual events. After creating a new Virtual Event Experience, you will be prompted to pre-populate the experience. Brightcove recommends choosing No thanks.

Much of the experience customization is done using a configuration file. The configuration file is added to the template as part of the experience header. After creating a new Virtual Event Portal Experience, the first thing that should be done is to add the configuration file to the template.

The configuration file controls some of the general template options and is also used to:

  • Customize the header
  • Customize the footer
  • Configure the event schedule
  • Customize the event speakers
  • Customize the event sponsors

The process to add videos to the experience is similar to other Gallery Portal Experiences although the Virtual Event template does expect the collections to be organized in a specific way. For information, see Organizing Video Collections in a Virtual Event Experience.

Adding the configuration file to the template

The Virtual Event Portal Experience requires a configuration file be included with the site. The configuration file includes site settings, layout options, event schedule, speakers and other content that will be displayed. After creating a new Virtual Event Experience, adding the configuration file is the first thing that should be done. For information on how to add the configuration file, see Adding the Configuration File to the Experience.

Configuring the header

The header displays at the top of all the pages and provides a way to brand and add social icons to the site. For more information on configuring the header, see Customizing the Header for a Virtual Event Experience.

Configuring the footer

The footer displays at the bottom of all the pages and provides a way to brand and add social icons and links to the site. For more information on configuring the footer, see Configuring the Footer for a Virtual Event Experience.

Configuring the event schedule

The event schedule is typically used to list the schedule of live events. For more information on configuring the event schedule, see Configuring the Event Schedule.

Customizing the event speakers

The speakers section is used to highlight the event speakers. For more information on customizing the event speakers, see Customizing the Event Speakers.

Customizing the event sponsors

The sponsors section is used to highlight the event sponsors. For more information on customizing the event sponsors, see Customizing the Event Sponsors.
