Creating a Virtual Event Experience

In this topic you will learn about creating a Virtual Event Experience.

Gallery provides a Virtual Event template which can be used to create Virtual Event Experiences for hosting and delivering virtual events. After creating a new Virtual Event Experience, you will be prompted to pre-populate the experience. Brightcove recommends choosing No thanks.

The process to add videos to the experience is similar to other Gallery Portal Experiences although the Virtual Event template does expect the collections to be organized in a specific way. For information, see Adding Videos to a Virtual Event Experience.

Adding the configuration file to the template

The Virtual Event Portal Experience requires a configuration file be included with the site. The configuration file is added to the template as part of the experience header and controls some of the general template options and is used to configure and customize several sections of the site:

  • Event schedule
  • Speakers
  • Sponsors

Brightcove recommends adding the configuration file right after creating a new Virtual Event Portal Experience. For information, see Adding the Configuration File to the Experience.

Customizing the layout

The Gallery Site Editor is used to customize the layout of pages. A logo, menu links, welcome message and the page footer can all be customized using the Gallery Site Editor. For more information on customizing the layout, see Customizing the Content of Virtual Event Experience Pages.

Configuring the event schedule

The event schedule is typically used to list the schedule of live events. For more information on configuring the event schedule, see Configuring the Event Schedule.

Customizing the event speakers

The speakers section is used to highlight the event speakers. For more information on customizing the event speakers, see Customizing the Event Speakers.

Customizing the event sponsors

The sponsors section is used to highlight the event sponsors. For more information on customizing the event sponsors, see Customizing the Event Sponsors.
