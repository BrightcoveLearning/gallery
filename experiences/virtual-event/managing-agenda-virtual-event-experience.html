Managing the Event Agenda

In this topic you will learn how to upload and configure the event agenda for a Virtual Event Experience.

The agenda module supports uploading an external .csv agenda file so that all of the agenda items don't have to be manually entered into the agenda module. Uploading an agenda file is how the event agenda is initially created. After an agenda is uploaded, it can be edited using the agenda module.

Note: For information on creating an agenda file for upload, see Creating an Event Agenda File.

Uploading an agenda file

Before importing your existing agenda into the agenda module, the agenda file should be exported to a .csv file. The agenda module can only import a .csv file.

Using the Gallery Site Editor, you can upload and configure the event agenda. To upload an agenda file to a Virtual Event Experience, follow these steps.

  1. Click the experience name to open the Gallery Site Editor.
  2. In the left navigation, click SITE FEATURES > Agenda.
  3. Click Upload Agenda.
  4. Browse and select the agenda file to upload.
  5. Confirm the agenda is listed on the My agenda page.

If any errors occur during the agenda import, a message will be displayed. After the import, the Status will be DEACTIVATED. Agendas with a status of DEACTIVATED will not be displayed in the Virtual Event Experience.

Adding a session

To add a new session, follow these steps.

  1. Click on the agenda to open the agenda editor.
  2. Click on an open time slot.
  3. Enter the session information.
  4. Click Save.

If any errors occur during the agenda import, a message will be displayed. After the import, the Status will be DEACTIVATED. Agendas with a status of DEACTIVATED will not be displayed in the Virtual Event Experience.

Activating and deactivating an agenda

The agenda Status determines whether or not the agenda will be displayed in the associated Virtual Event Experience.

  • ACTIVATED - The agenda will be displayed in the associated Virtual Event Experience
  • DEACTIVATED - The agenda will not be displayed in the associated Virtual Event Experience

To activate an agenda, click on it and then click Activate. To deactivate an agenda, click on it and click Deactivate. Note that after activating/deactivating an agenda, the experience doesn't need to be republished.
