Integrating Pigeonhole with a Virtual Event Experience

In this topic you will learn how to integrate Pigeonhole with a Virtual Event Experience.

Pigeonhole can be used to easily add live Q&A to one or more videos in a Virtual Event experience. Typically, a new Pigeonhole is created for the live event and then sessions can be created for each video you want to add Q&A to. Once the sessions are created, iframe embed code is associated with each video using a Video Cloud custom field. The embed code will cause the Pigeonhole Q&A pod to display next to the video player for chosen videos.

Creating a new Pigeonhole event

You should create a new Pigeonhole for your live event and then add sessions for each video that will have Q&A enabled.

  1. Login to your Pigeonhole account.
  2. Click + Add Pigeonhole and click your account to use.
  3. Enter an Event name and Passcode.
  4. Enter Start and End dates for the event. These dates control how long the Pigeonhole will be active.
  5. Select a Timezone and Custom Branding option (if applicable).
  6. Click Continue.
  7. Now that the event is created the Agenda item will be selected in the left navigation. Click Add session and click Create new.
  8. Select a Session Type of Q&A.
  9. Enter a Session name.
  10. Enter Start and End dates and times.
  11. Configure the Q&A Settings as appropriate.
  12. Click Add Q&A. Confirm the session was created.
  13. Click More settings in the left navigation.
  14. Click the Integrations tab at the top of the page.
  15. Scroll down to the Others section.
  16. Locate Brightcove Gallery Embed and click Integrate.
  17. (Optional) If your Gallery Experience will be secured using SSO, select Allow automatic attendee sign-in using SAML attributes. This will retrieve the user’s name and email address and automatically sign them in on Pigeonhole Live when the Pigeonhole Live widget loads. If this is not checked, users will appear as anonymous inside Pigeonhole. Click Save.
  18. On the Brightcove Gallery Embed dialog, select your Q&A session from the dropdown list.
  19. Click Get embed code.
  20. Click on the embed code and copy it to the clipboard. This is the code that will be embedded on the Gallery experience.
  21. Click Close.

Creating a custom field

In order for the Virtual Event template to know which Pigeonhole to display for a video (if any), a custom field is used. To create a custom field, follow these steps.

  1. Login to Video Cloud Studio.
  2. In the navigation header, click ADMIN and then Video Fields. Note that you must be an account administrator to access to the ADMIN menu.
  3. Create a new custom field named Chat Embed.
  4. Confirm that the internal name is chat_embed.
  5. Set the Type to be Text.
  6. Save the custom field.

For information on creating custom fields in Video Cloud, see Creating Custom Metadata Fields.

Adding Q&A to a video

To have a Pigeonhole appear next to a video in the Virtual Event template, follow these steps.

  1. Open the Media module.
  2. Click on a video to open the video properties.
  3. Locate the CUSTOM FIELDS section and click Edit.
  4. Paste the Pigeonhole embed code into the Chat Embed custom field.
  5. Click Save.
