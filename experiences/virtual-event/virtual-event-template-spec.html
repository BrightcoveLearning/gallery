Virtual Event Template Specifications

This topic provides an overview of the content requirements for the Virtual Event template.

Video and Audio

Video and audio recommendations are listed below.

Video

Format MP4
Codec H.264, H.265
Bitrate 2x the bitrate of the highest rendition
Frame rate Constant, usually between 15 - 60 frames per second
Aspect ratio Your video aspect ratio will be preserved. An aspect ratio of 16:9 is common.

Audio

Codec AAC (Advanced Audio Codec)
Bit rate Constant, 128 kbps
Channels Stereo

For recommendations on best practices for recording, exporting, uploading and publishing your videos to ensure optimum playback, see Best Practices: The Video Creating, Transcoding, and Uploading Workflow.

Images

Various images are used throughout the Virtual Event template. Image recommendations are listed below.

Background Image

The image size depends on the breakpoints the customer is trying to support, more often than not 1920x1080 works, but the image should be optimized so it's not gigantic for best performance.

Header Logo

The header logo image will be proportionally scaled down to 178px wide.

Agenda

Default image size is 320px x 180px. Images will be cropped to this size.

Speakers

Speaker images should have a max width of 200px. Images will not be resized. By default, the template CSS configures the speaker images with a border-radius of 50% which will make the images circular. The speaker bio displayed below the speaker is limited to approximately 3 lines of 15 words.

Sponsors

The sponsor images will be resized to 100px wide.

Fonts

The template is optimized to work with the Roboto Condensed font. This font can be assigned using the Style menu inside of the Gallery interface. Other fonts can be used but may require changes to CSS files. For information about using custom fonts, see Using Custom Fonts in a Portal Experience.

Schedule

The schedule is typically used to list the schedule of live events. There is no limit to the number of items that can appear in the schedule. The schedule can display the session name, speakers and date and time for the session. A countdown timer will appear next to the session and a View button will appear when the session is live. The session description is not displayed on the page itself but can be displayed using custom JavaScript. For information on configuring the event schedule, see Configuring the Event Schedule.
