Uploading and Customizing the Event Agenda

In this topic you will learn how to upload an customize the event agenda for a Virtual Event Experience.

You may have created an event agenda in Microsoft Excel or another spreadsheet application. The agenda module supports uploading an external .csv agenda file so that all of the agenda items don't have to be manually entered into the agenda module.

Note: A sample agenda template file is located here.

Import file requirements

In order to import your existing agenda into the agenda module, the agenda should have been created in a spreadsheet application (i.e. Microsoft Excel, Apple Numbers or Google Sheets. These applications allow the data to be exported to a .csv file. The agenda module can only import a .csv file.

The agenda module looks for the existence of certain column names. These columns will be mapped to the appropriate fields in the agenda module. At a minimum, the following fields should exist in your agenda .csv file:

  • name - Session name
  • start_date - Session start date
  • end_date - Session end date
  • start_time - Session start time
  • duration - Session duration

The following columns are optional, but if they exist in your .csv file, they will also be imported into the agenda module:

  • desc - Session description
  • speaker1_first - Speaker first name
  • speaker1_last - Speaker last name
  • speaker2_first - Speaker 2 first name
  • speaker2_last - Speaker 2 last name
  • track - Value that identifies the track (i.e. technical, sales, marketing)
  • session_type - Session type
  • video _id - Video ID
  • breakout_url - Breakout room URL (for breakout sessions)
