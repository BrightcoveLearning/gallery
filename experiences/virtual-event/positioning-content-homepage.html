Positioning Content on the Home Page

In this topic you will learn how to position objects that appear on the Virtual Event Experience homepage.

Speaker images and a short bio can be added to the Virtual Event homepage.

The speaker information is customized by editing the speakers object in the configuration file.

  • name - Speaker name
  • description - Speaker information that displays below their image
  • image_url - Speaker image
  • link_url - Destination URL for the image (may link to speaker LinkedIn page for example)

The speakers object in the configuration file also has parameters to control where in the homepage the speaker information is displayed.

  • name - Speaker name
  • description - Speaker information that displays below their image
