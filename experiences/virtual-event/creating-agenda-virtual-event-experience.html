Creating an Event Agenda File

In this topic you will learn how to create an event agenda file that can be uploaded to initially populate an an online agenda for a Virtual Event Experience.

Typically, the event agenda is initially created in a third-party tool, for example, Microsoft Excel or Google Sheets. The agenda module supports uploading an external .csv agenda file so if the agenda was initially created in a third-party tool, all of the agenda items don't have to be manually re-entered into the agenda module.

Note: A sample agenda template file can be downloaded here.

Column naming requirements

When creating the initial agenda file, you can download a sample agenda file and edit it or use an existing agenda file you may have created. The agenda module looks for the existence of specific column names. These columns will be mapped to the appropriate fields in the agenda module. Other columns that may be in your agenda file will be ignored by the agenda module. In order to upload an agenda file, the following columns are required:

  • name - Session name
  • start_date - Session start date (MM/DD/YYYY format)
  • start_time - Session start time
  • timezone - Timezone for start_time (using zone ID from the IANA time zones)
  • duration - Session duration in minutes

The following columns are optional, but if they exist in your agenda file, they will also be imported into the agenda module:

  • description - Session description
  • speaker1_first - Speaker first name
  • speaker1_last - Speaker last name
  • speaker1_email - Speaker email address
  • speaker2_first - Speaker 2 first name
  • speaker2_last - Speaker 2 last name
  • speaker2_email - Speaker 2 email address
  • track - Value that identifies the track (i.e. technical, sales, marketing). If blank, then it will default to our special All setting which means it's a session that spans all tracks.
  • session_type - Session type (keynote, session, other)
  • video _id - Video ID
  • related _label - Label for the related link that will appear in the Agenda module
  • related_link - URL link to session handout, slides, etc.
Note: While the agenda upload supports up to two speakers, additional speakers can be added to a session using the agenda builder.

When you are ready to import your agenda into the agenda module, export it as a .csv file. The agenda module can only import a .csv file. For information on how to upload your agenda file into the agenda module see Uploading and Customizing the Event Agenda.
