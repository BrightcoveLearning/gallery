Managing Event Speakers

{{ page.description }}

Event speakers are managed using the Agenda module. To access the Agenda module, follow these steps:

  1. Click on a Virtual Event Experience to open the Gallery Site Editor.
  2. Click SITE FEATURES > Agenda.
  3. In the left navigation, click Speakers. By default, the Event Speakers link is selected displaying all speakers for the current event.

Clicking the All Speakers link will display all the speakers that have been assigned to all Virtual Event Experiences. The speaker list can be sorted by clicking on a column header name.

Creating a speaker

To create a new speaker, follow these steps.

  1. Click Create Speaker.
  2. Enter the speaker First Name and Last Name.
  3. Enter the speaker Email.
  4. Enter the speaker Title and Company.
  5. Enter the speaker Bio.
  6. For the Headshot image, click Upload Image or the upload icon and browse and select a speaker image.
  7. Click Save.

Editing a speaker

To edit a new speaker, follow these steps.

  1. Click on a speaker to open the Edit Speaker panel.
  2. Make needed changes.
  3. Click Save.
