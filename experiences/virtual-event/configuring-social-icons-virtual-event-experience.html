Configuring the Social Icons for a Virtual Event Experience

In this topic you will learn how to customize the social icons that appear in Virtual Event Experience pages.

Social icons can be added to Virtual Event experiences so that viewers can easily access your social media sites. By default, social icons will be displayed in the header and footer of all site pages:

  • Facebook
  • Instagram
  • Twitter
  • LinkedIn
  • YouTube

The icons that appear can be edited/removed by editing the social object in the configuration file.

  • link_url - Destination URL for the image
  • image_url - Social image to appear in the header

To remove a social icon, remove the appropriate entry from the configuration file.
