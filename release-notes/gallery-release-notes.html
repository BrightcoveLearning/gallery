Brightcove Gallery™ Product Updates

This topic outlines the updates that have been made to the Brightcove Gallery application.

Release notes are also available for:

  • Gallery Template Release Notes

Auto-Play Video Setting

4 Sep 2019

A new Auto-Play Video setting has been added and can be used to control playback behavior of videos on detail pages. For information, see Customizing the Player and Lead Form Settings for a Portal Experience.

Redesigned Gallery Navigation

27 Sep 2018

The navigation menu when editing portal experiences has been redesigned to be more intuitive and consistent with the in-page experience editing environment. For information, see Using the Site Editor to Customize a Portal Experience.

Search Option Added to Gallery Homepage

1 Aug 2018

A Search box has been added to the Gallery homepage providing the ability to search through the list of experiences. When searching, the following properties will be searched:

  • Title
  • Description
  • ID
  • Template
  • URLs

Adding Videos to an In-Page Experience

18 Jul 2018

When adding videos to an in-page experience, there is now an option to use a playlist reference ID. By using a playlist reference ID, the videos in the experience can easily by changed by re-assigning the reference ID to another playlist using the Media module. The experience doesn't need to be edited and republished when you need to change the videos that appear in the experience. For information, see Using a playlist reference ID.

Changes to Lead Forms

15 May 2018

Gallery now uses the Audience module to create and manage lead forms. Audience will collect the lead data and provides a data export option. For information, see Using a Lead Form with a Portal Experience.

Live Event In-Page Template

14 May 2018

Gallery now provides a Live Event In-Page template which can be used to deliver live events as part of an In-Page Experience. For information, see Creating a Live Event In-Page Experience.

Portal Experience Site Settings UI Updates

13 Apr 2018

The Site Settings UI has been updated to remove the video related settings from the Custom dialog to a new Video settings dialog. For more information, see Customizing the Video Settings for a Portal Experience.

Marquee Template Setting

30 Mar 2018

A new featured video behavior setting has been added to Marquee template. For more information, see Customizing the Marquee Template Settings.

Custom Header and Footer Option

30 Mar 2018

It is now possible to include the default header and footer when using a custom header and footer. For more information, see Configuring custom header/footer HTML.

In-Page Experience 3Play Components

1 Feb 2018

3Play Media interactive transcript components can now be added to an in-page experience. This allows viewers to see a video transcript as a video is played. For more information, see Adding Components to an In-Page Experience.

In-Page Experience Chat Components

17 Jan 2018

Chat components can now be added to an in-page experience. For more information, see Adding Components to an In-Page Experience.

Customized Access Control Page

28 Nov 2017

The access control page can now be styled so that it is more closely branded with the rest of the site. For more information, see Styling the access control page.

Stylized 404 Pages

6 Nov 2017

404 pages will now use the styling configured with the site (headers and footers) so they are more closely branded with the rest of the site.

Live Event Template - Background Image Options

6 Nov 2017

The Live Event template now provides options for how background images are handled. For more information, see Customizing the Live Event Template Settings.

Collection Metadata

6 Nov 2017

For all templates (except Marquee) you can now add collection descriptions and it will serve as the metadata description for the category page (for browsers sharing the collection). For more information, see Adding Video Cloud Videos to a Portal Experience.

API Key No Longer Required

29 Sep 2017

The API Key is no longer required and has been removed from the Settings page.

Multiple In-Page Experiences on Same Page

15 Aug 2017

Gallery now supports having multiple instances of the same in-page experience on the same page. This allows you to create one experience and then dynamically pass in a playlist or list of videos to display. For more information, see Using Dynamic Video Overrides in an In-Page Experience.

Event Experience Now Supports using Reference ID

9 Aug 2017

When configuring the live event state for an event experience, the live video asset can now be specified using a reference ID. Using a reference ID provides more flexibility as the reference ID can be assigned to the live video asset without having to make changes to the experience. For more information, see Creating the Live Event State for an Event Experience.

Additional In-Page Experience Player Setting

27 Jul 2017

A new player setting has been added to enable an in-page experience to remain in the playing state when a video completes. This can be used to display the player endscreen. For more information, see Configuring the Settings for an In-Page Experience.

Dynamic Video Overrides in an In-Page Experience

27 Jul 2017

It is now possible to dynamically override the videos associated with an in-page experience by modifying the embed code. For more information, see Using Dynamic Video Overrides in an In-Page Experience.

In-Page Experience Training Now Available

21 Jul 2017

On-demand training for creating in-page experiences is now available. For more information, see Training on Demand: Building In-Page Experiences using Gallery.

YouTube Video Support for In-Page Experiences

11 Jul 2017

It is now possible to use YouTube videos in an in-page experience. For more information, see Adding Videos to an In-Page Experience.

Use Site Description in Search Engine Results

10 Jul 2017

Gallery now provides an option to use site description in search engine results. This will use the site description in the <meta> description tags, including the Facebook and Twitter meta tags. For more information, see Configuring Search Engine Optimization Settings for a Portal Experience.

Catalogue Template

28 Jun 2017

Gallery now provides a Catalogue template. For more information, see Overview of Gallery Portal Templates.

In-Page Experiences

31 May 2017

Gallery now provides the option of creating In-Page Experiences. For more information, see Overview: Creating, Editing and Publishing In-Page Experiences.
