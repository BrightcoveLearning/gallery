Getting Started

Brightcove Gallery™ makes it easy to create and publish videos experiences so viewers can consume your content wherever they may be. Two types of experiences can be created:

  • Mobile App - Create a device application so your videos can be viewed inside of a native iOS or Android application (learn how...)
  • Web Application - Create a video website so your videos can be viewed at a specific URL on desktop and mobile browsers (learn how...)
